# Contributing Guide

Thanks for contributing! This repository focuses on front-end workflows for the AI audio/video assistant.

## Scope
- UI, UX, and interaction logic in `src/`
- API client usage and integration boundaries
- i18n strings and UI copy

## Setup
```bash
npm install
npm run dev
```

## Code Style
- TypeScript only
- 2-space indentation, double quotes, no semicolons
- Use existing components in `src/components/`

## Tests
- `npm run test`
- Add tests for new shared components or behavior changes

## Commits
- Use Conventional Commit prefixes (e.g. `feat: ...`, `fix: ...`, `chore: ...`)

## Pull Requests
- Provide a short summary
- Include screenshots for UI changes
